# InvoiceLab 📄

> 专业发票生成器 — 免费、美观、无需注册

## 🌐 在线使用
**https://tyr1105.github.io/invoicelab/**

## ✨ 特性
- 📝 **专业发票**: 创建美观的专业发票
- 🎨 **3套模板**: Modern/Classic/Creative 三种风格
- 📄 **PDF导出**: 一键生成高质量PDF文件
- 👥 **客户管理**: 保存客户信息，快速复用
- 💰 **多币种**: 支持CNY/USD/EUR/HKD/GBP/JPY
- 🔄 **发票历史**: 查看所有发票，状态管理
- ⚙️ **自定义设置**: 默认信息、税率、发票编号格式
- 🔒 **本地存储**: 所有数据保存在浏览器，无需上传

## 🎯 适用场景
- 自由职业者给客户开票
- 小型企业发票管理
- 服务商费用结算
- 个人兼职收入记录
